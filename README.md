# Event Management System

## Overview

Event Management System is a web-based application developed using PHP, HTML, CSS, JavaScript, and MySQL. The system helps users browse events, make bookings, manage registrations, submit feedback, and process payments. Administrators can manage events, bookings, users, and overall system operations through a dedicated dashboard.

## Features

### User Features

* User Registration and Login
* Browse Available Events
* Event Booking
* Online Payment Processing
* View Booking History
* Submit Feedback and Reviews

### Admin Features

* Secure Admin Login
* Manage Events
* Manage Bookings
* Manage Users
* View Reports
* Monitor System Activities

## Installation

1. Install WAMP/XAMPP.
2. Copy the project folder to the `www` or `htdocs` directory.
3. Import the MySQL database.
4. Update database credentials in `config.php`.
5. Start Apache and MySQL.
6. Open the project in your browser.
